# 'Nombre de la app'

Solución del reto 7 de CoAfina 2025: "Índices del futuro: Tierra y Mar Argentino". Grupo Neotropical 3.0

## a

This is a Streamlit-based hackathon data/AI application for analyzing environmental and economic indices.

## Features

- **Live Analysis**: Real-time data visualization with interactive line charts
- **AI Model**: Interactive interface for AI-powered predictions and analysis

## Installation

1. Clone this repository:
```bash
git clone https://github.com/intent-andy/indices-del-futuro_CoAfina2025.git
cd indices-del-futuro_CoAfina2025
```

2. Install the required dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the Streamlit application:
```bash
streamlit run app.py
```

The application will open in your default web browser at `http://localhost:8501`.
